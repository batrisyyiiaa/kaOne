# kaOne
AI pocket CFO for Malaysian individuals and small business owners. Built for UM Hackathon 2026 using ILMU-GLM-5.1.

## What it does

- Smart Tax Forecaster – predicts LHDN tax based on your income
- Subscription Kill Switch – finds unused subscriptions and asks if you want to cancel
- Transaction Splitter – learns whether a transaction is personal or business after you tell it once
- AI Chat Analyst – answers questions about your finances in plain language
- Inventory & Budget Tracking – for small business owners who need to track stock

## Built for

- Individuals 
- Small business owners (home bakers, freelancers, gig workers)

## Tech stack

- HTML / CSS / JavaScript
- ILMU-GLM-5.1 (Z.AI model)
- LocalStorage for data persistence

## How to run

1. Clone this repository
2. Open index.html in a browser
